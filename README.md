## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```sh
    git clone https://github.com/enock-niyonsaba/Divine-Portfolio.git
    ```

2. Navigate to the project directory:
    ```sh
    cd Divine-Portfolio
    ```

3. Install the dependencies:
    ```sh
    npm install
    ```
